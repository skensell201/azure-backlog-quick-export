# Backlog Quick Export — выгрузка бэклога в CSV/Excel

Расширение для Azure DevOps Server 2022. Добавляет на страницу бэклога команды вкладку **Export**, с которой можно одним кликом выгрузить **весь текущий уровень бэклога** в CSV или Excel — с иерархией и суммами по задачам.

## Установка (выполняет администратор коллекции)

1. Скачать файл `.vsix` из релизов: **https://github.com/skensell201/azure-backlog-quick-export/releases** (последняя версия).
2. В Azure DevOps вверху справа нажать значок **🛍 Marketplace → Manage extensions**.
3. **Upload new extension** → выбрать скачанный `.vsix`.
4. У загруженного расширения нажать **Install** и выбрать коллекцию.

## Как пользоваться

1. Открыть **Boards → Backlogs** нужной команды.
2. Перейти на вкладку **Export** (рядом с *Backlog* и *Analytics*).
3. В списке **Уровень бэклога** выбрать уровень (Epics / Features / Stories и т.п.).
4. Нажать **Скачать CSV** или **Скачать Excel** — файл скачается в браузер.

## Что в выгрузке

- **Весь уровень бэклога** целиком, без ручного выделения элементов.
- **Иерархия**: дочерние элементы вложены под своим элементом, заголовок (Title) сдвинут отступом по уровням.
- **Колонки**: Order, ID, Work Item Type, Title, State, Value Area, Iteration Path, Tags, Sum of Task Original Estimate, Sum of Task Completed Work.
- **Суммы по задачам** (Sum of Task…) считаются по вложенным Task.
- **Ссылки на рабочие элементы**: в **Excel** колонка **ID** — кликабельная ссылка, открывает рабочий элемент в браузере; в **CSV** ссылка лежит в отдельной колонке **URL**.

## Примечания

- Язык вкладки (русский/английский) определяется автоматически по языку интерфейса Azure DevOps.
- CSV — в кодировке UTF-8 (корректно открывается в Excel, кириллица не «ломается»).
